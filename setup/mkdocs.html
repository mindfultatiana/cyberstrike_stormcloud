Installing¶

  1. Install Python 3.7+

  2. Install the packages: 

    pip install mkdocs
pip install markdown
pip install pymdown-extensions
pip install mkdocs_custom_fences

  3. Serve the website

    python -m mkdocs serve

  4. Build the website

    python -m mkdocs build
